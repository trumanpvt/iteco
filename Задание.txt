Необходимо разработать React компонент для отображения данных в виде, представленном на вложенном изображении (строки могу раскрываться, внутри просто уникальный текст), 
добавить возможность поиска по последним 4-м цифрам карты, сортировку по дате c возможностью выбора направления сортировки. 
Компонент должен получать данные путем REST запроса файла JSON.

В качестве сборщика Webpack, верстка HTML5, CSS без фреймворков, JS только React и компоненты для асинхронных запросов на выбор.
